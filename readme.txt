== Description ==
This plugin allow you to show some animations before loading full content of your website.
 

== Installation ==

To install, simple extract the 'animated preloader' folder into your 'wp-content/plugins/' directory. Once extracted, you must activate the plugin within the WordPress Site Admin 'Plugins' section. Then go to Widget section and you'll find out the animated preloader plugin.
